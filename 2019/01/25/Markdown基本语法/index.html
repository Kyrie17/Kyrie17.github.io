Markdown基本语法

[TOC]

概述

Markdown是一种纯文本格式的标记语言。通过简单的标记语法，它可以使普通文本内容具有一定的格式。

优点

  1. 因为是纯文本，所以只要支持Markdown的地方都能获得一样的编辑效果，可以让作者摆脱排版的困扰，专心写作。
  2. 操作简单。比如:WYSIWYG编辑时标记个标题，先选中内容，再点击导航栏的标题按钮，选择几级标题。要三个步骤。而Markdown只需要在标题内容前加#即可。

缺点

  1. 需要记一些语法。
  2. 有些平台不支持Markdown编辑模式。

语法

1. 标题

在想要设置为标题的文字前面加上来表示
一个#是一级标题，两个#是二级标题，以此类推 可至六级标题

效果如下:

这是一级标题

这是二级标题

这是三级标题

这是四级标题

这是五级标题
这是六级标题

2. 字体

  • 加粗
    **这是加粗的文字**
  • 斜体
    *这是倾斜的文字*
  • 斜体加粗
    ***这是倾斜加粗的文字***
  • 删除线
    ~~这是加删除线的文字~~

效果如下：

这是加粗的文字
这是倾斜的文字
这是倾斜加粗的文字
这是加删除线的文字

注意：
上述*,~等符号应在英文状态下，且包含文字前后不需要空格
