选择

二分只有上述两种情况：

  • 找大于等于给定数的第一个数
  • 找小于等于给定数的最后一个数

下面给一个对比的例子（leetcode34）： Given an array of integers nums sorted in ascending order, find the starting and ending position of a given target value. If target is not found in the array, return [-1, -1]. Follow up: Could you write an algorithm with O(log n) runtime complexity?

思路：分别找出第一次出现的位置和最后一次出现的位置，即分别对应模板1和模板2。
